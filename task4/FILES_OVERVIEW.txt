WORKFLOW PHASES
═════════════════════════════════════════════════════════════════

Phase 1: SCANNING (5-30 minutes)
├─ Walks entire folder tree
├─ Finds all files
├─ Reads file sizes
└─ Output: List of all files

Phase 2: ANALYSIS (< 1 minute)
├─ Categorizes by extension
├─ Calculates MD5 hashes
├─ Identifies duplicates
├─ Finds large files
└─ Output: Detailed statistics

Phase 3: REPORTING
├─ Displays file summary
├─ Shows category breakdown
├─ Lists duplicate groups
├─ Identifies large files
└─ Calculates recovery potential

Phase 4: DRY-RUN PREVIEW
├─ Shows all operations
├─ Groups by category
├─ No files touched
└─ Requires your review

Phase 5: APPROVAL
├─ Waits for user decision
├─ Type "yes" to proceed
├─ Type anything else to cancel
└─ No automatic proceeding

Phase 6: EXECUTION
├─ Creates output folders
├─ Copies files to categories
├─ Maintains file metadata
├─ Logs all operations
└─ Shows progress

Phase 7: REPORTING
├─ Generates report file
├─ Summarizes results
├─ Lists errors (if any)
└─ Provides next steps

SUPPORT & TROUBLESHOOTING

Issue: Script is slow

→ Progress updates every 50 files
→ Let it finish

Issue: "Folder not found"
→ Use absolute path: python organize_mess.py /full/path/to/folder
→ Check spelling

Issue: "Permission denied"
→ Close applications using the folder
→ Run with administrator privileges

Issue: Not enough disk space
→ Need space = original folder size
→ Use different output drive
→ Free up space before running
